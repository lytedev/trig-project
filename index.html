Triangle Solver by @lytedev

Supply at least 3 of the following. If more than 3 are given, the first 3 will be used. This could change the outcome of your triangle, so it is recommended to use only 3 variables. None of them may be 0 or less, and no angle may be 180 or greater. Angles are measured in degrees. If one set of three variables fails, try using a different set of 3.

If you aren't sure which variables you're given or which ones to input, take a look at this image explaining which variables represent different sides and angles in a triangle.
